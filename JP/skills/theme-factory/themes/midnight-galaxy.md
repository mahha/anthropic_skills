# Midnight Galaxy（ミッドナイト・ギャラクシー）

深いパープルと神秘的なトーンで構成した、ドラマチックで宇宙的なテーマ。強い印象を残したいプレゼンに向きます。

## カラーパレット

- **ディープパープル（Deep Purple）**: `#2b1e3e` - リッチな暗色ベース
- **コズミックブルー（Cosmic Blue）**: `#4a4e8f` - 神秘的な中間トーン
- **ラベンダー（Lavender）**: `#a490c2` - 柔らかなアクセント
- **シルバー（Silver）**: `#e6e6fa` - 明るいハイライトとテキスト
